# Dev Portfolio

MY developer portfolio built using HTML, CSS and Vanilla JavaScript.

## Overview

This project represents the first version of my professional portfolio.  
It is built without frameworks to demonstrate strong fundamentals in:

- DOM manipulation
- Modular JavaScript architecture
- State management
- Dynamic rendering
- UI structure and responsive design

A future version will migrate this project to React to improve component reusability and scalability.

## 🛠 Tech Stack

- HTML5
- CSS3 (modular structure)
- Vanilla JavaScript (ES Modules)

## Goals

- Demonstrate strong JavaScript fundamentals
- Showcase professional project presentation
- Prepare architecture for future React migration
